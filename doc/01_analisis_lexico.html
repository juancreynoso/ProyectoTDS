Análisis Léxico

Descripción

El análisis léxico se encarga de transformar la entrada de texto en una secuencia de tokens. Estos tokens son unidades atómicas que alimentan al parser.

Herramientas

  • Flex para el scanner
  • Integración con Bison a través de yylval

Ejemplo de regla en el lexer

{ID}    { yylval.sval = strdup(yytext); return TOKEN_ID; }
