Introducción

Este proyecto tiene como objetivo implementar un compilador para un lenguaje diseñado a medida, siguiendo las fases clásicas de compilación: análisis léxico, análisis sintáctico, análisis semántico, generación de código intermedio, generación de código objeto y optimización.
