Introducción

Este proyecto tiene como objetivo implementar un compilador para un lenguaje diseñado a medida, siguiendo las fases clásicas de compilación: análisis léxico, análisis sintáctico, análisis semántico, generación código intermedio, generación código objeto y optimización.

El compilador está implementado utilizando las herramientas estándar Flex y Bison, que permiten la generación automática del analizador léxico y sintáctico respectivamente, garantizando robustez y eficiencia en el proceso de compilación.

División del Trabajo

En esta primera etapa del proyecto, ambos integrantes del equipo trabajamos de manera colaborativa en todas las fases implementadas. No hubo una división específica de tareas, sino que se adoptó un enfoque de programación en conjunto que permitió:

  • Mejor comprensión compartida del código y arquitectura
  • Detección temprana de errores mediante revisión continua
  • Decisiones de diseño consensuadas entre ambos integrantes
  • Aprendizaje mutuo de las herramientas Flex y Bison

Metodología de Trabajo

  • Sesiones de desarrollo conjunto: Implementación colaborativa utilizando Live Share (Extensión de VSCode)
  • Testing compartido: Ambos integrantes aportamos diferentes casos de prueba
  • Documentación paralela: Registro de decisiones y problemas durante el desarrollo
